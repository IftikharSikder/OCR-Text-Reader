# Optical Character Recognition — Flutter

A Flutter app that extracts text from images using Google ML Kit — works offline, supports live camera and gallery import.

## Features

- Live camera scanning
- Gallery image import
- On-device OCR (no internet required)
- Copy to clipboard & share

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | Flutter 3.x |
| OCR Engine | Google ML Kit Text Recognition |
| Image Picker | `image_picker` package |
| State Management | Bloc |
